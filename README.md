# 🎨 AI Virtual Air Drawing Application

## 📌 Project Description
AI Virtual Air Drawing Application is a computer vision project developed using Python, OpenCV, and MediaPipe. This application allows users to draw in the air using their index finger through a webcam without touching any physical surface.

---

## 🚀 Features

- ✋ Real-time hand detection
- ☝️ Index finger tracking
- 🎨 Air drawing using finger movement
- 🔴 Multiple color options
- 🧽 Eraser mode
- 🗑️ Clear canvas option
- 💾 Save drawing as image
- 📷 Real-time webcam display

---

## 🛠️ Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy

---

## ⌨️ Controls

| Key | Function |
|-----|----------|
| R | Red Color |
| G | Green Color |
| B | Blue Color |
| P | Purple Color |
| E | Eraser |
| C | Clear Canvas |
| S | Save Drawing |
| Q | Quit Application |
